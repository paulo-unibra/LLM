   A simulação agora mostra os cálculos REAIS que LLMs fazem:
   
   📊 Embeddings (Etapa 2.5):
   - Vetores de 4 dimensões (simula os 12.288D do GPT)
   - Exemplo: [0.742, 0.389, -0.156, 0.903]
   - Palavras similares = vetores próximos
   
   🔗 Mecanismo de Atenção (Etapa 3.5):
   - Matriz 3x3 mostrando relações entre tokens
   - Query · Key^T / √d (fórmula dos Transformers)
   - Visualização com cores: forte (azul), média (roxo), fraca (cinza)
   
   📈 Softmax & Probabilidades (Etapa 4.5):
   - Distribuição de probabilidades para o próximo token
   - e^(logit/T) / Σe^(logit/T)
   - Mostra top 5 candidatos com probabilidade de cada um
   - Temperature controla aleatoriedade

📚 PONTOS TÉCNICOS PARA DESTACAR:

1. **Embeddings:**
   "Cada palavra vira um vetor de 12.288 números!
    Palavras com significados similares têm vetores próximos.
    É como mapear linguagem em espaço matemático."

2. **Atenção:**
   "Esta é a MÁGICA dos Transformers!
    Cada token 'presta atenção' nos outros.
    Query·Key^T calcula relevância matemática.
    Revolucionou a IA em 2017!"

3. **Softmax:**
   "O modelo NÃO sabe a resposta certa.
    Ele calcula PROBABILIDADE para cada palavra possível.
    Temperature controla criatividade vs determinismo.
    É por isso que respostas podem variar!"

4. **Operações:**
   "175 bilhões de parâmetros.
    Cada token = ~175 mil multiplicações de matrizes.
    Uma frase simples = bilhões de operações!
    Por isso GPUs são essenciais."
