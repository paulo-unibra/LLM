📖 Como Funciona:

1. Tokenização: Texto é dividido em tokens (palavras/pedaços)
2. Embedding: Tokens viram números (vetores 12.288D)
3. Atenção: Modelo calcula relações entre tokens
4. Processamento: 12 camadas transformam os vetores
5. Softmax: Converte logits em probabilidades
6. Geração: Escolhe próximo token baseado em probabilidade

🧮 Cálculos Matemáticos Reais:

Embedding: token → vetor[12288]
Atenção: Q·KT/√d + Softmax
Softmax: ex/Σex (normaliza probabilidades)
Temperature: logits/T (controla aleatoriedade)
Custo: ~$0.002 por 1000 tokens (GPT-3.5)

🎓 Conceitos:

LLM: Large Language Model (Modelo de Linguagem Grande)
Token: Unidade básica de texto (~4 caracteres)
Prompt: Instrução/pergunta enviada ao modelo
Temperature: Criatividade da resposta (0-1)
Context Window: Quanto texto o modelo "lembra"
Parâmetros: Pesos aprendidos durante treinamento
